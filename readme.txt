== Description ==

This plugin adds a pack of shortcodes to enhance your site. You can easily add buttons, alerts, columns and more without coding knowledge! 

= Shortcodes Includes =
* Alert
* Button
* Column
* Tab
* Toggle
* Dropcap
* Hightlights
* **More to come**

[Check out the demo](http://demo.theme-junkie.com/daily/shortcodes/).

== Installation ==

**Through Dashboard**

1. Log in to your WordPress admin panel and go to Plugins -> Add New
2. Type **theme junkie shortcodes** in the search box and click on search button.
3. Find Theme Junkie Shortcodes plugin.
4. Then click on Install Now after that activate the plugin.
5. Go to add new Post or Page, you will see a **Add Shortcode** button beside the media button.

**Installing Via FTP**

1. Download the plugin to your hardisk.
2. Unzip.
3. Upload the **theme-junkie-shortcodes** folder into your plugins directory.
4. Log in to your WordPress admin panel and click the Plugins menu.
5. Then activate the plugin.
5. Go to add new Post or Page, you will see a **Add Shortcode** button beside the media button.
